# VTU Web App — Virtual Top-Up Platform

The VTU Web App is a modern fintech-inspired application that allows users to purchase airtime and data seamlessly. It focuses on providing a smooth, secure, and user-friendly transaction experience from start to finish.

---

## Live Demo

https://vtu-web-app-gilt.vercel.app

---

## Overview

This project simulates a real-world Virtual Top-Up (VTU) platform used for:

- Buying airtime
- Purchasing mobile data
- Managing transaction flows

It is designed to reflect how fintech products handle multi-step transactions, confirmations, and success states.

---

## Features

### Airtime Purchase Flow

- Enter phone number and amount
- Transaction confirmation screen
- Success page with transaction feedback

---

### Data Purchase Flow

- Service Grid
  - Select network/provider
- Data Purchase Page
  - Choose data plan
- Data Confirmation Page
  - Review before purchase (in progress / planned)

---

## Transaction Experience

- Multi-step flow:
  - Input → Confirm → Success
- Clean user feedback at each stage
- Designed to mimic real fintech UX patterns

---

## UI/UX Highlights

- Clean, modern fintech-style interface
- Clear transaction states (important for trust)
- Mobile-first responsive design
- Simple and intuitive navigation

---

## Tech Stack

- Frontend: React
- Routing: React Router
- Build Tool: Vite
- Styling: TailwindCSS
- State Management: React Hooks
- Future Integration: Backend APIs for real transactions

## Installation & Setup

1. Clone the repository:

   git clone https://github.com/your-username/vtu-app.git

2. Navigate into the project:

   cd vtu-app

3. Install dependencies:

   npm install

4. Start development server:

   npm run dev
